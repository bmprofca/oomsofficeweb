# Tech Stack

## Frontend

- React (functional components + hooks)
- Tailwind CSS
- Framer Motion (`motion`, `AnimatePresence`)
- `react-icons` (multiple icon packs)

## Data / Networking

- `fetch` for most requests
- `axios` for specific flows (especially multipart uploads)

## Feedback / UX

- `react-hot-toast` for success/error/loading notifications

## Auth Mechanism

- Auth values are read from localStorage and sent in headers via `getHeaders()`
